# ssp-client
ssp-client - клиентская часть системы SSP (Secured SeamlessTCP Proxy). 
С помощью протокола SOCKS5 клиент преобразует входящие соединения в
SeamlessTCP протокол для дальнейшего туннелирования.

## Конфигурация
`host` - адрес для привязки клиента

`servers` - адреса серверов или зеркал для подключения

`retries` - максимальное количество попыток для восстановления соединения

`timeout` - максимальное время ожидания ответа при передаче данных

`log_level` - уровень логирования
